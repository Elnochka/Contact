Весь проект находится на GitHub/Elnochka/Contact
Чтобы развернуть SQL базу нужно exampl.sql положить в папку bin, установленного SQL сервера
и выполнить команду source exampl.sql или -f dumpfile -U nameuser -d database в созданной БД.
Для запуска приложения необходимо, чтобы на ПК был установлен Apache Tomcat 7.0.
Если проект будет запускться не с локального компьютера, необходимо внести изменения
в файл application.properties.
